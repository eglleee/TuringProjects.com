# Module_1_capstone_project

This is a program that will organize your finances! Every month you can enter your income, expenses, savings, categorize them, and see various reports. The data can be saved for future analysis.

## Features

- Input and categorize income, expenses, and savings.
- Generate reports and summaries of your expenses and savings.
- Save data for future reference and analysis.
- User-friendly menu-driven interface.

## Getting Started

### Prerequisites

- Python 3.x installed on your system.

### Installation

1. Clone this repository:
   ```
   git clone https://github.com/eglleee/TuringProjects.com.git
   ```
2. Navigate to the project directory:
   ```
   cd Module_1_capstone_project
   ```

## Usage

To run the program, execute the following command:
```
python Main_file.py
```

You will be shown a menu and prompted to enter options such as "a", "b", "c", "d", or "f". Based on your selection, further options will appear to guide you through the program's functionality.

## Example

Here is a quick example of how the program works:

1. Run the program:
   ```
   python Main_file.py
   ```
2. Select an option from the menu:
   - "a" to Edit new period finances
   - "b" to Record existing period as finalized
   - "c" to Get a report of current period
   - "d" to Check savings
   - "f" to exit the program
